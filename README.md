# LMS Project

This project contains both the frontend and backend for the Learning Management System.

## Project Structure

- **/frontend**: React/ client side.
- **/backend**: Node.js/Express API.

## Getting Started

### 1. Backend Setup

1. `cd backend`
2. `npm install`
3. Create a `.env` file
4. `npm start`

### 2. Frontend Setup

1. `cd frontend`
2. `npm install`
3. `npm start`
